What is WasteZero?

Data Flow & Access Control

Citizens

  • Create pickup requests (→ Open)
  • Track status updates
  • Cannot assign volunteers

Volunteers

  • Accept Open requests (→ Accepted)
  • Complete pickups (→ Completed)
  • Coordinate via messaging

Admins

  • Monitor all platform activity
  • Control user activation
  • Generate and export reports
